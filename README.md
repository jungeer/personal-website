# 个人主页

基于 React + TypeScript + Tailwind CSS 构建的个人主页，展示个人信息、技术栈、项目经历等。

## 特性

- 🎨 现代化的UI设计
- 🌓 支持亮色/暗色主题切换
- 📱 完全响应式布局
- ✨ 平滑的动画效果
- 🔍 动态导航标题
- 🔝 智能回到顶部
- 📋 便捷的复制功能

## 技术栈

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
- Lucide React

## 主要功能

- 个人信息展示
- 技术栈展示（支持查看全部）
- 开源项目展示
- 项目经历展示（带详情弹窗）
- 工作经历展示
- 主题切换
- 回到顶部
- 动态导航标题

## 本地开发

```bash
npm install
npm run dev
```
